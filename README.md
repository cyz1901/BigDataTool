# **BigDataTool**

------

BigDataTool 是一个开源的分布式大数据管理工具，目前还是早期版本，只支持Hadoop的部署。



## 介绍

------

BigDataTool主要由基于Vue.js的UI和基于gpc和SpringBoot的后端分布式服务组成。
